# DevValue — The VibeCoder's Cognitive & Value Tracker

Tracks the true cost of AI-era software engineering per git branch:
Cost = (Human Focus Time × Hourly Rate) + (AI Agent Token Cost)

## Features
- PTA Flow Detection — intelligent idle timeout that recognizes thinking time
- Token Sniffing — parses Claude Code logs for real AI compute costs
- Branch Binding — all costs tracked per git branch
- Real-time Status Bar — live time, branch, and cost display
- Dashboard — per-branch cost breakdown with token usage details
